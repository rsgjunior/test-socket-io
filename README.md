# test-socket-io

Aplicação de teste implementando Web Sockets utilizando a biblioteca socket.io

## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/rsgjunior/test-socket-io
```

Entre no diretório do projeto

```bash
  cd test-socket-io
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
